This plugin supports responsive grid view to load products using ajax and without full page refresh. It also allows to customize text and background colors of the heading.

== Installation ==

1. Upload and extract the zip file downloaded to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'WCPT' page from dashboard and create new shortcode to add into any page content (Premium/Pro version only)
4. Configure widgets "Category and Product Tab" from Appearance > Widgets and assign to page sidebar.
